Seguí estos pasos una sola vez. Después compartís este archivo HTML y todos usan la misma hoja.

Paso 1 — Abrí script.google.com y creá un proyecto nuevo.

Paso 2 — Pegá el código del archivo biblioteca-apps-script.gs que te pasé.

Paso 3 — Hacé clic en Implementar → Nueva implementación. Tipo: Aplicación web. Ejecutar como: Yo. Acceso: Cualquier persona.

Paso 4 — Copiá la URL que aparece y pegala acá abajo.

Paso 5 — Guardá y compartí este archivo HTML con tu equipo.
